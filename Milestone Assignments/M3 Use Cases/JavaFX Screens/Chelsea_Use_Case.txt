Use Case: Player visits Town

Scope: M.U.L.E. Game

Primary Actor: Player

Stakeholders and Interests: 
	1. Player: interested in visiting Town
	2. Other Game Players: interested in other players state in the game 

Preconditions: 
	1. A player must be selected/created
	2. A player must select a race and color 

Postconditions:
	1. Player visits the pub
	2. Player visits land office
	3. Player visits the store

Main Success Scenario: 
	1. Player requests new game initialization
	2. Player starts game
	3. Player enters Town 

Extensions: 
	1. Enter Town (step 4)
		*alternate flow: leaves Town
